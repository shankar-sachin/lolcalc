### Hosting

- Download the `tar.gz` or `zip` source code file from the latest release
- Open the source root in Terminal or Powershell
- Make sure Python 3 + PIP is installed
- Type `pip install -e .`
- Then once it downloads the build files, type `lolcalc` to launch

### Commands

- `lolcalc` launches the interactive viewer
- `--version` shows you the version you have installed
- `lolcalc "expression"` gives you the answer without the viewer
- There are a couple of easter eggs
- Enjoy!
